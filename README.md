# SpaceNovaX V7.1 Mining Engine Core

채굴기를 우선 완성하기 위한 Mining Engine V1.0 핵심 버전입니다.

## 핵심 정책
- 24시간 = 24 SPNX Point
- 1시간 = 1 SPNX Point
- 서버 기준 시간 계산
- Fleet 1명 = +1%, 최대 +100%
- 35% 채굴 보상 풀 기준 반감기
- Claim 서버 검증
- 중복 채굴 방지
- Sandbox 5분 테스트 모드

## 추가 기능
- MiningEngine 컴포넌트
- 서버 기반 실시간 진행률
- 속도/보너스/반감기/보상 표시
- 관리자 Mining Monitor 탭
- Sandbox ON/OFF
- 채굴 강제 Reset
- 채굴 풀 잔량 / 오늘 채굴량 표시
