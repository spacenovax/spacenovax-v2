# SpaceNovaX NOVA-X1 Galaxy War V1

실행 가능한 우주함대 세로 슈팅 게임 V1입니다.

포함 기능:
- NOVA-X1 기체
- 모바일 드래그 이동/자동 발사
- 적 편대/엘리트 적/적 탄환
- 보스 전함과 체력바
- 크리스탈/무기강화/실드/수리
- 점수/HP/스테이지/폭발
- 보스 +10 SPNX 규칙
- 크리스탈 1,000개 +5 SPNX 규칙
- 일일 최대 20 SPNX 로컬 제한
- 게임 설명창

실행:
1. npm install
2. npm run dev

주의: 실제 SPNX 지급은 서버 검증을 붙여야 합니다. 현재 localStorage는 데모용입니다.
